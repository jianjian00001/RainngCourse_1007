## 技术栈

### 前端Vue.js

- ElementUI
- axios

### 后端SpringBoot

- 持久层: Mybatis Plus
- 授权认证: 手写
- 会话: Spring Session + Redis
- 日志: AOP + MongoDB

### 数据库

- MySQL(MariaDB): 核心业务
- Redis: Session、新闻、配置信息
- MongoDB: 业务日志
